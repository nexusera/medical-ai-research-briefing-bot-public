# 工具使用真相 (Tool Use Ground Truth)

技能唯一依赖且授权的外部数据入口：`WebSearch`

## WebSearch 工具使用指南
1. **多层查准逻辑**:
   - 必须优先从权威信息源（arXiv, medRxiv, PubMed, PMC, Nature, NEJM）抓取。
   - 遇到学术界黑话或新型专有缩写，必须在 Query 中加上原名词的变体展开形。
2. **时间窗硬性卡点**:
   - 如果用户查询每日进展，搜索工具调用强制加上时间范围条件（例如过去 24 小时或 48 小时）。
   - 如果 48h 颗粒度返回为 0 且用户意图为新闻，扩大时间窗至 72h 甚至 7 天补救。
3. **相关模式软性回溯 (Related Strategy)**:
   - 如果用户的 Query 包含“对比基线”、“近年代表作”、“脉络”等词态，必须从 `WebSearch` 参数上撤掉 48h/72h 时间约束。
   - 转移到“按相关性与来源权重排序”，且针对大龄基准模型（>3年前），在返回时打上 `[Seminal / Baseline]` 标签。
