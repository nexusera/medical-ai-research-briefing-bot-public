# 医疗调研结果输出格式

根据具体的临床决策需求或细节要求匹配相应的格式。

## 1. ⚡ 决策快报 (Executive Briefing)
*目标读者：决策者、忙碌的临床医生。*
- **重点**：核心突破、临床影响及“最终结论 (Bottom Line)”。
- **结构**：
  ```markdown
  # 🏥 [方向] 决策快报
  **核心结论 (BLUF)**: [1-2句话直接回答临床价值]
  **置信度**: [高/中/低]
  
  **关键发现**:
  - [研究成果 1]
  - [临床意义]
  
  **主要局限**: [最核心的一个缺点]
  **建议行动**: [下一步的具体建议]
  ```

## 2. 📰 标准晨报 (Standard Morning Brief)
*目标读者：日常研究追踪、同行讨论。*
- **重点**：技术脉络、优劣对比及弱点识别。**要求列出所有发现的论文。**
- **结构**：(详见 SKILL.md 中的默认模板)
  ```markdown
  # 🏥 [方向] 每日医学综述
  ## 🔗 论文列表 (Full Paper List)
  1. [论文标题 A](链接)
  2. [论文标题 B](链接)
  
  ## 🔬 深度剖析
  ### [论文标题 A]
  [...演进关系、核心贡献、优缺点分析...]
  ```

## 3. 🔬 深度学术综述 (Comprehensive Academic) - 默认格式
*目标读者：协议设计、批判性评论或论文撰写。*
- **重点**：详尽的方法论、偏倚分析及多源证据交叉验证。
- **结构**：
  ```markdown
  # 🏥 深度医学综述: [方向]
  ## 摘要 (Abstract)
  [概述研究问题、方法、主要发现及结论]
  
  ## 检索策略 (Search Methodology)
  [使用的数据库、布尔查询串、日期范围]
  
  ## 技术深度调研 (In-depth Findings)
  ### [技术方向 A]
  [基于 Methods 部分的深度分析与量化数据对比]
  
  ## 临床证据质量评价 (Evidence Quality)
  [偏倚风险评估、FDA/NIH 相关监管政策对标]
  
  ## 讨论与结论
  [技术脉络总结、未来可增强方向的详细推导]
  
  ## 参考文献 (References)
  [详尽的索引列表，包含所有在 48h/3d 窗口内发现的相关研究，带直接链接]
  ```

## 4. 🧪 科研草稿 (Working Notes)
*目标读者：内部持续研究、草稿准备。*
- **重点**：原始发现、未解决的问题及检索日志。
- **结构**：
  ```markdown
  # [方向] — 科研笔记
  **状态**: [进行中 / 已完成 / 需更多证据]
  **最后更新**: [日期]
  
  ## 已知重要事实
  - [具体发现] — [来源] — [可靠性/置信度]
  
  ## 待查问题 (Open Questions)
  - [ ] [尚未解决的疑问或需要进一步验证的点]
  
  ## 检索日志 (Search Log)
  [尝试过的关键词、有效的渠道、无效的尝试]
  ```
